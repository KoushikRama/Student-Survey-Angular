Welcome to the Student Survey Application

Please choose an option:

  • Fill Student Survey
  • View All Surveys

This is a full-stack web application built using Angular 17 and Spring Boot. It allows prospective students to complete a campus visit survey, and administrators to view, update, and manage all submitted survey records.

Project Team

  • Koushik Rama
  • Abhijith Devulapally

Features

  • Fill and submit the prospective student survey form
  • Store survey results in a MySQL database
  • View all submitted surveys in a structured table
  • Edit or delete any existing survey entry
  • Form validation and error handling for all fields
